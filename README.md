# books-api

Catch my latest book recommendations on my website

> Built using Django Rest Framework

## Available Routes

| #  | Route  | Description  | Request |
|---|---|---| ---|
| 1.  | `api/`  |  Lists all the books | GET, PUSH|
| 2.  | `api/<id>`  | List specific book details  | GET, PUT, DELETE |
